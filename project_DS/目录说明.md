# project_DS

这个目录是项目自定义的开发规范库，主要给人类和 agent 提供补充约束与流程说明。

## 职责边界

- `AGENTS.md`：Codex 原生自动加载文件，负责项目摘要、强约束、子代理 SOP。
- `.agents/skills/`：Codex 原生 repo skill 目录，负责技能触发与调度能力。
- `project_DS/`：项目自定义规范文档库，负责沉淀详细规则、流程说明、设计标准。

## 当前目录约定

- `specification/`：项目级架构、编码、UI 规范。
- `workflows/`：项目执行流说明，包括任务规划流与自动开发流。
