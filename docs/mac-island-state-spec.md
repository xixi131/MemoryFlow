### Todo status-update persistence contract (code-derived)

- Backend contract: controller forwards this request to `TodoService.updateTaskStatus`, which maps `completed=true` to `status=completed` + `completed_at=now`, and `completed=false` to `status=todo` + `completed_at=null`, then persists via `todoTaskMapper.updateById`. Evidence: `back-end/src/main/java/com/memoryflow/controller/TodoController.java:131`, `back-end/src/main/java/com/memoryflow/service/TodoService.java:307`, `back-end/src/main/java/com/memoryflow/service/TodoService.java:310`, `back-end/src/main/java/com/memoryflow/service/TodoService.java:313`, `back-end/src/main/java/com/memoryflow/service/TodoService.java:316`.
- Local schema reference: persisted fields are `todo_tasks.status` and `todo_tasks.completed_at`, with status enum `('todo','completed')`. Evidence: `back-end/src/main/java/com/memoryflow/config/TodoSchemaMigrationRunner.java:47`, `back-end/src/main/java/com/memoryflow/config/TodoSchemaMigrationRunner.java:53`, `back-end/src/main/java/com/memoryflow/config/TodoSchemaMigrationRunner.java:57`, `back-end/src/main/java/com/memoryflow/entity/TodoTask.java:22`, `back-end/src/main/java/com/memoryflow/entity/TodoTask.java:51`.
